# R code for reproducing the analyses from article "Stochastische Überlagerung in der Regionalstatistik - Relevanz und Auswirkungen künstlicher Messfehler"

## 01_replication Prenzel

R script for reproducing the analysis from section 4.1

The analysis builds on:

P. Prenzel: “Kann man das überhaupt messen?” – Der Bedarf an detaillierten räumlichen Bevölkerungsdaten in der Wirtschaftsgeographie; in: Valentiner DS, Goldberg K (Eds.) Zensus 2022 – Volkszählung zwischen Recht, Politik und Sozialwissenschaften, Nomos, Ch. 4, pp.65–86, https://doi.org/10.5771/9783748915980-65

Data not included. For data sources see [here](https://github.com/mamoeh/ckm_regio_2026/blob/main/01_replication_Prenzel/data/data_sources_01.md).

Data not included. For data sources see [here](https://github.com/mamoeh/ckm_regio_2026/blob/main/03_replication_Gril/data/data_sources_03.md).
